# 常量管理规范

最后更新：2026-04-02

## 原则

- 跨页面共享常量进入 `src/constants/**`
- 页面域内常量放在对应页面目录的 `constants.ts` 或语义常量文件
- 纯展示文案优先进入 locale，不留在业务组件里

## 允许的常量类型

- 状态映射
- 表单选项
- 固定阈值
- 主题 token
- 列配置默认值

## 禁止项

- 魔法字符串散落在多个组件
- 为了偷懒把一切都塞进 `data.ts`
- 在业务组件里继续扩散十六进制色值

## 当前重点

- `results` 高频文案与状态映射
- `configuration` 选项与状态映射
- `access` 权限类型与页签定义
